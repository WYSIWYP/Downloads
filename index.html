WYSIWYP download files

MusicXML files are downloaded to your Downloads folder. PDF files contain sheet music in WYSIWYP format and are opened into a new brower tab via your default PDF viewer app (e.g., Adobe Acrobat Reader). From there, the file may be downoaded or printed.

Once a MusicXML is converted to WYSIWYP by SNapp, the format of the sheet music can be adjusted with user Preferences. Then a new customized PDF file can be saved from SNapp. This new file can then printed via a PDF viewer.
